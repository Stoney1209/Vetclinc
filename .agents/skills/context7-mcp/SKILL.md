Context7 MCP — Plantilla de Skill (ejemplo)

Nombre: context7-mcp
Descripción: Proporciona documentación y ejemplos de APIs/librerías en tiempo real.
Versión: /org/project
Entrada: query de usuario (lib/software/docs)
Salida: ejemplos de código, referencias y versión citada

Uso básico:
- Resolver library-id para el nombre de la librería
- Buscar docs con query-docs usando el ID obtenido
- Responder citando versión y proporcionando ejemplos relevantes
